## âœ¨ Architecture

**MVVM Pattern**
- **Models**: `ETFData.swift`, `ETFDataManager.swift`
- **Views**: `ContentView.swift`, `ETFDetailView.swift`
- **App**: `DividendETFTrackerApp.swift`

**Data Flow**
1. ETFDataManager loads JSON from bundle
2. Decodes into ETF structs
3. Published to Views via @StateObject
4. Views observe changes and update UI

**No External Dependencies**
- Pure SwiftUI
- Built-in JSON decoding
- Native iOS APIs only
